私の消費タイプ SBTI

SBTI消費タイプテストとは？

SBTIはSpending Behavior Type Indicator、つまり消費行動をタイプ分けして見る楽しいテストです。何を買うときに迷うのか、どこには気持ちよくお金を使うのか——人それぞれ違う消費習慣をいくつかのタイプに映し出します。MBTIのように気軽に楽しめるコンテンツです。

消費習慣は自分を映す鏡

同じお金でも、ある人は経験に、ある人はモノに、ある人は将来のための貯蓄に使います。消費の優先順位には、その人の価値観や好みが自然とにじみ出ます。普段の自分の買い物習慣を正直に思い浮かべて答えると、自分でも気づかなかった消費傾向に出会えます。

進め方

12の簡単な質問に、普段の消費習慣を思い浮かべながら答えてください。1〜2分ほどで終わります。最後には自分の消費タイプと特徴の説明に出会え、結果は友だちと共有することもできます。
